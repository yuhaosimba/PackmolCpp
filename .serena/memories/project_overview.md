PackmolCpp is a Linux-based Packmol development repository centered on the Fortran reference implementation for molecular packing. Core build system is CMake + Ninja with Fortran sources under src/ and the CLI entrypoint in app/packmol.f90. Tests live under tests/ with unit, component, gencan, and regression layers. Repository-local skills document the Fortran baseline and migration workflow. Python is used for inventory generation and some test checkers. Pixi is the recommended development entrypoint and provisions cmake, ninja, python 3.11/3.12, gfortran, and julia.
